# Nextera Systems

Next.js project using SCSS modules with a clean, component-based structure.

## Getting started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Structure

- `src/app` - App Router pages and global styles
- `src/components/layout` - shared layout components
- `src/components/sections` - page sections
- `src/components/ui` - small reusable UI elements
